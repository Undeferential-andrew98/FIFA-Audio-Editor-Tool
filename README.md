## 🛠️ Managing Audio Files

The application interface lists the audio categories on the left.

### Importing New Sounds
1. Select the category you wish to change, such as "Chants" or "Commentary."
2. Click the "Import" button.
3. Choose the audio file from your computer. Note that the file must be in the correct format, such as .wav or .ogg, to ensure the game engine recognizes it.
4. Click "Apply" to save the changes to your local archive.

### Exporting Existing Sounds
1. Select the desired audio track from the list.
2. Click "Export."
3. Choose a save location on your computer.
4. The tool saves the file in a standard format for you to edit in external audio software.

## ⚠️ Troubleshooting

If you encounter issues, review these common fixes:

* The tool does not find the game path: Ensure you have selected the folder that contains the game executable (.exe file).
* Audio does not play in-game: Check if the file format matches the original game file requirements. Rebuild the audio archive using the "Rebuild" button in the menu.
* Application freezes: If the tool stops responding during export or import, do not force close it immediately. Give the program extra time to process high-resolution audio files.
* Game crashes: If the game crashes after applying mods, use the "Clean" feature to remove custom changes and reset the audio archives to their original state.

## 📑 File Structure
The tool creates a small folder in your local directory for temporary storage. Do not delete these files while the application runs. If you need to uninstall the tool, simply delete the folder containing the executable and the local temporary files. 

## 🛡️ Safety and Best Practices
Always create a backup of your audio data before making changes. Locate your game folder, find the "Data" or "Audio" subfolder, and copy it to a safe place. If you make a mistake, you can copy these files back to restore the original sound.

This application does not modify your FIFA account or online status. However, modding files may cause issues with online play. Disable the tool or restore original game files before playing in official online modes to avoid conflicts with anti-cheat software.
